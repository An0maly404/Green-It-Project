Welcome to the Green IT Project

The Green IT Project is designed to help users measure and reduce their digital carbon footprint. By understanding the environmental impact of your digital activities, you can make informed decisions to minimize your ecological footprint.

Our Eco-Friendly Measures

  • Efficient Code: Optimized JavaScript and CSS to reduce resource usage.
  • Next-Gen Images: Using WebP format for faster loading and reduced bandwidth.
  • Lazy Loading: Images are loaded only when needed to save energy and bandwidth.
  • Minimalist Design: A clean and simple interface to reduce unnecessary resource consumption.
  • Carbon Footprint Awareness: Tools to calculate and track your digital carbon footprint.

Technical Features

  • Secure Authentication: Passwords are hashed using SHA-256, and sessions are managed with JWTs.
  • SQLite Database: Lightweight and efficient database for storing user data and scores.
  • Serverless Deployment: Hosted on Vercel for scalability and eco-friendly cloud infrastructure.
  • API Integration: Seamless communication between the frontend and backend using Axios.
  • Admin Dashboard: Manage users and monitor scores with role-based access control.
